# Health Insight AI Automations

Shared BADM 350 Module 3.2 automation workspace for Health Insight AI, a chatbot that helps users understand symptoms and possible next steps.

## What Lives Here

- `my-project-context/` contains the project context files created in Module 3.1
- Module 3.2 automation outputs will be saved here as they are built

## Team Workflow

1. Pull the latest changes before starting work.
2. Work on separate files when possible.
3. Commit and push often.
4. Pull again before editing a file someone else may have changed.
